JavaScript Index Deletion

Learn how to properly delete indices from arrays and array-like objects

Concept

In JavaScript, we don't delete "indices from objects" - we delete properties from objects or elements from arrays.

When working with arrays, we have several methods to remove elements by index:

1. Using splice()

The most common way to remove elements from an array by index.

let arr = [10, 20, 30, 40, 50];
arr.splice(2, 1); // Removes 1 element at index 2
// Result: [10, 20, 40, 50]

2. Using filter()

Creates a new array without the element at a specific index.

let arr = [10, 20, 30, 40, 50];
arr = arr.filter((_, index) => index !== 2);
// Result: [10, 20, 40, 50]

3. Using delete operator

Removes the element but leaves the array with an empty slot (not recommended).

let arr = [10, 20, 30, 40, 50];
delete arr[2]; // arr[2] is now undefined
// Result: [10, 20, undefined, 40, 50]

Important Notes

Arrays vs Objects: JavaScript arrays are actually objects with numeric keys. When you delete an array element, you're actually deleting a property with a numeric name.

Performance: splice() modifies the original array and may be faster for large arrays than filter() which creates a new array.

Empty slots: Using the delete operator creates empty slots in arrays, which can lead to unexpected behavior when iterating.
